# Design Tokens Mapping

This document lists the visual variables and token maps defined in `src/styles/tokens.css` that act as the styling source of truth.

---

## The Design Tokens

### 1. Colors
- **Black Backgrounds**:
  - `--color-black`: `#0a0a0a` (Main background)
  - `--color-black-soft`: `#141414` (Card panel surfaces)
- **Brand Colors**:
  - `--color-red`: `#d40000` (Core brand red)
  - `--color-red-hover`: `#c10004` (Hover active states)
- **Supporting Grays**:
  - `--color-white-muted`: `#b8b8b8` (Supporting descriptions)
  - `--color-steel-dark`: `#2a2a2a` (Card and inputs border)

### 2. Spacing Scale
- **`--space-xs`**: `8px`
- **`--space-sm`**: `16px`
- **`--space-md`**: `24px`
- **`--space-lg`**: `48px`
- **`--space-xl`**: `96px`

### 3. Typography
- **Families**:
  - `--font-display`: `'Montserrat', sans-serif` (Headings, titles, tech specs)
  - `--font-body`: `'Google Sans', 'Roboto', sans-serif` (Paragraph copy)
- **Sizes**:
  - `--font-size-xs`: `12px`
  - `--font-size-sm`: `14px`
  - `--font-size-base`: `16px`
  - `--font-size-md`: `20px`
  - `--font-size-lg`: `28px`
  - `--font-size-xl`: `40px`
  - `--font-size-2xl`: `56px`

### 4. Borders & Elevations
- **Radii**:
  - `--radius-card`: `12px`
  - `--radius-input`: `6px`
- **Elevations**:
  - `--shadow-medium`: `0 12px 32px rgb(0 0 0 / 60%)`
  - `--shadow-elevated`: `0 16px 40px rgb(0 0 0 / 50%)`
